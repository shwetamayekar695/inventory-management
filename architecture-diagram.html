End-to-End Request Flow

1 — Filter change
useFilters composable
User selects warehouse, category, status, or month. Shared ref updates instantly across all views.
reactive watch triggers
2 — View reacts
Vue View Component
watch() on filter refs calls loadData(). Sets loading = true, clears previous data.
calls api.js method
3 — HTTP request
api.js (Axios)
Builds URLSearchParams. Skips filters set to "all". Sends GET to localhost:8001/api/…
HTTP GET + query params
4 — Route handler
FastAPI endpoint
Receives optional query params. Passes in-memory list + params to apply_filters() and filter_by_month().
▼
5 — Filter
In-Memory Filtering
apply_filters(): warehouse exact match, category case-insensitive, status match. filter_by_month(): direct month or quarter→months expansion.
filtered list
6 — Validate
Pydantic serialisation
Filtered dicts validated against response_model. Fields typed, optional fields handled. HTTPException 404 if single-item not found.
JSON array response
7 — Store raw data
Vue ref update
Axios response.data written into raw ref (e.g. allOrders.value). loading = false. Error cleared.
computed re-evaluates
8 — Render
Computed + Template
Computed properties (totals, chart data, filtered lists) auto-recalculate. Vue patches only changed DOM nodes.
